# Pizzería Bauti - Gestión de Reservas

Este proyecto es una aplicación web para la gestión de reservas de la Pizzería Bauti.

## Características

- Formulario de reservas para clientes (`reservas.html`).
- Panel de administración para ver y gestionar reservas (`index2.html`).
- Almacenamiento local de reservas (LocalStorage).

## Uso

1. Abrir `reservas.html` para crear una nueva reserva.
2. Abrir `index2.html` para ver el listado de reservas y administrarlas (editar/borrar).

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
